## What is CosmicBotz?

CosmicBotz is a Telegram bot organisation focused on building premium, production-grade bots for anime and media communities. Every bot in this ecosystem is designed around one principle — eliminate manual work while delivering a polished, reliable user experience.

The organisation maintains a suite of interconnected bots, each handling a distinct part of the content pipeline: from indexing and search, to file storage and delivery, to channel management and community automation.

<br/>

---

## Projects

<br/>

### Auto Filter CosmicBotz
The flagship. A fully automated content indexing and delivery system for anime groups. Members search any title and receive a timed invite link to the channel — no admin involvement required.

<br/>

### Video Sequence Bot
A channel administration bot that groups uploaded video files by series, season, and episode — then posts them with inline quality selector buttons that link directly to file store entries.

<br/>

### Alisa File Store Bot
The file delivery backbone. Stores files, generates encoded share links, and handles batch delivery. Built on the CodeXBotz FileStore pattern with significant architectural improvements.
